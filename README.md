# Agent Match Screen Interface (prototype)

Prototype UI for MLS / Agent Match screens.

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm install
npm run build
```

## Static preview (SPA)

```bash
python3 spa_server.py --port 8090 --bind 0.0.0.0
```

Then open:
- http://localhost:8090/
- http://localhost:8090/home
